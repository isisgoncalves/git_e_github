Como publicar no GitHub

Na pasta que será feito o commit, do Visual Studio Code para o GitHub, abrir a opção "Terminal" no menu superior.

Abrir a opção "Novo Terminal"

Ao abrir o Terminal, uma janela no canto inferior será aberta. Então digitar os seguintes comandos:

git init função para indicar que a pasta foi inicializada com o GIT
git remote add (nome do diretório) (endereço do diretório no GitHub) Faz a conexão entre o diretório local com o diretório remoto
git branch -m main Devido a alteração da "branch" principal do GitHub não ser mais "Master", é necessário alterar o nome da nossa branch para "Main", assim como está no GitHub
